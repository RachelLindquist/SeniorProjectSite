To get started using the application, you'll first need to sideload it into your oculus quest see here for a detailed guide on how to do that. The site also shows how to access side loaded content from the headset.

If you do not want to run the application through the actual headset, you can also run it through a Windows computer. Make sure to download the package and plug in your turned on headset using an Oculus Link cable. Afterwards click on the .exe file to launch the application on your computer. If you have problems, please look here for a more detailed explanation.
